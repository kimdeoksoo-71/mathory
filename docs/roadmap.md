### 6-1. 좌측 사이드바

| 항목 | 상태 | 비고 |
|------|------|------|
| 사이드바 레이아웃 (확장/축소 토글) | ✅ | 축소 시 아이콘만 표시 |
| 사이드바 헤더: Mathory 로고 + 토글 아이콘 | ✅ | 로고: 명조체 계열 |
| 섹션 1: + 새 문제 / 🔍 검색 | 🔄 | |
| 섹션 2: 폴더 관리 | ✅ | 새 폴더 추가, 문항→폴더 이동 |
| 폴더 선택 → 우측 작업창에 문항 목록 표시 | ✅ | |
| 섹션 3: 최근 문항 (최근 편집 순) | ✅ | |
| 최근 문항 클릭 → 우측에 문제+해설 보기 | ✅ | |
| 문항 hover → ⋯ 메뉴 (이름변경, 편집, 폴더변경, 삭제) | ✅ | 삭제 위 구분선 |
| 하단: Google 로그인/로그아웃 (위에 구분선) | ✅ | |
| 축소 모드 아이콘: 토글, +, 🔍, 📂, ≡, 👤 | ✅ | |
