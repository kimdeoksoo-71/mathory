# 에디터 기능 명세

## 에디터 레이아웃

Split View 구조로 왼쪽에 편집, 오른쪽에 실시간 미리보기.
```
┌─────────────────────────────────────────────┐
│  MathToolbar  [f] [√] [^] [_] [∫] [Σ] ...   │
├──────────────────────┬──────────────────────┤
│                      │                      │
│   MarkdownEditor     │   EditorPreview      │
│   (CodeMirror 6)     │   (KaTeX + MD)       │
│                      │                      │
│   raw_text 편집       │   실시간 렌더링          │
│                      │                      │
└──────────────────────┴──────────────────────┘
```

## 에디터 핵심 기능

### 1. Markdown + LaTeX 혼합 편집
- 일반 텍스트는 Markdown 문법
- 인라인 수식: `$...$`
- 블록 수식: `$$...$$`
- CodeMirror 6의 Markdown 언어 지원 + 커스텀 LaTeX 하이라이팅

### 2. 실시간 미리보기
- 에디터 입력 → debounce(300ms) → react-markdown + rehype-katex 렌더링
- 스크롤 동기화: 에디터 스크롤 위치에 맞춰 미리보기도 이동

### 3. 수식 편집기 툴바 (MathToolbar)

커서 위치에 LaTeX 템플릿을 삽입하고, `{}` 안에 placeholder를 배치하여 Tab 키로 이동.

| 버튼 | 라벨 | 삽입 텍스트 | 카테고리 |
|------|------|------------|----------|
| 분수 | f | `\frac{▫}{▫}` | 기본 |
| 제곱근 | √ | `\sqrt{▫}` | 기본 |
| 거듭제곱 | x² | `^{▫}` | 기본 |
| 아래첨자 | x₂ | `_{▫}` | 기본 |
| 정적분 | ∫ | `\int_{▫}^{▫} ▫ \, dx` | 미적분 |
| 합 | Σ | `\sum_{▫}^{▫} ▫` | 미적분 |
| 극한 | lim | `\lim_{▫ \to ▫} ▫` | 미적분 |
| 괄호 | () | `\left( ▫ \right)` | 괄호 |
| 절댓값 | \|\| | `\left\| ▫ \right\|` | 괄호 |
| 행렬 | [ ] | `\begin{bmatrix} ▫ \end{bmatrix}` | 행렬 |
| 부등호 | ≤ | `\leq` | 기호 |
| 부등호 | ≥ | `\geq` | 기호 |
| 무한대 | ∞ | `\infty` | 기호 |
| 화살표 | → | `\to` | 기호 |

> `▫` = Tab으로 이동 가능한 placeholder. CodeMirror의 snippet 기능 활용.

### 4. 수식 모드 자동 감지
- `$` 입력 시 수식 모드 진입 표시 (배경색 변경 등)
- `$$` 입력 시 블록 수식 모드
- 수식 모드 안에서만 MathToolbar 활성화 (선택사항)

## 블록 에디터

각 문제/풀이는 여러 블록으로 구성. BlockEditor가 블록 목록을 관리.

### 블록 조작
- 블록 추가 (하단 + 버튼)
- 블록 삭제
- 블록 순서 변경 (Phase 4에서 드래그앤드롭)
- 블록 타입 변경 (text ↔ choices ↔ hint 등)

### 블록 타입별 에디터 동작
- **text**: 기본 Markdown + LaTeX 에디터
- **choices**: 선택지 전용 UI (①~⑤ 자동 번호, 각 선택지에 LaTeX 사용 가능)
- **image**: 이미지 업로드 UI (Phase 4)
- **hint**: text와 동일하나 렌더링 시 접기/펼치기 UI
